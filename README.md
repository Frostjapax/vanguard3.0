Para rodar o otimizador diretamente no seu **Termux** sem a necessidade de baixar ou criar arquivos manualmente no armazenamento, basta copiar e colar o comando abaixo:

Pré-requisitos
Antes de iniciar, certifique-se de que você possui:

O aplicativo Termux instalado no seu celular.

Conexão ativa com a internet para efetuar o carregamento correto do script.

Para funções avançadas (como a Otimização Geral via Wi-Fi), ative previamente a Depuração por Wi-Fi nas Opções de Desenvolvedor do seu Android.

⚠️ Avisos Importantes
Uso Consciente: Algumas funções, como a liberação de 120 FPS e alterações profundas no sistema, podem elevar a temperatura do dispositivo. Recomendamos o uso moderado.

Responsabilidade: As modificações aplicadas são executadas por sua conta e risco. Faça sempre um backup das suas configurações importantes antes de realizar ajustes avançados.
